# 为微型社区 LaraVue 提供后端 API 接口，前端项目请前往 https://github.com/zmecust/laravue-frontend

## 运行环境

- Nginx 1.13.0
- PHP 7.0+
- Mysql 5.6+
- Redis 3.0+

## 安装

- git clone https://github.com/zmecust/laravue-backend.git
- composer install
- cp .env.example .env
- 配置 sendloud 邮件服务
- php artisan migrate
