# A forum api create by Laravel

## 运行环境

- Apache
- PHP 7.0+
- Mysql 5.6+
- Redis 3.0+

## Install

- git clone https://github.com/zmecust/laravue-backend.git
- composer install
- cp .env.example .env
- php artisan migrate
